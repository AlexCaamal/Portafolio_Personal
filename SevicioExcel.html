Macros en Excel

- Creacion de Formularios en Excel.
- Conocimiento en ciertas formulas de Excel.
- Utilización de Macros.

Excel es una excelente aplicación para poder almacenar, organizar y guardar información. Muchas veces lo utilizan como base de datos por el manejo de información y no es común en que tenga problemas de conexión. Ya que al tener una base de datos se enfrentan problemas de conectividad tanto del mismo dispositivo y de internet.

Una de las funcionalizades que excel maneja es el uso y creación de Macros (un conjunto de comandos que se almacena en un lugar especial de Excel de manera que están siempre disponibles cuando se necesite para ejecutar). Se pueden hacer secuencias repetitivas que una persona suele hacer, tales como, un conjunto de sumas, registro de datos, ordenamiento etc.

Se puede utlizar para almacenar información de suma importancia, utlizando botones, textbox labels etc.
